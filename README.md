# 🚀 20-Day JavaScript + DOM Learning Journey

## 📌 Overview

This repository documents my **20-day journey to learn JavaScript fundamentals and DOM manipulation** by building small projects daily.

The focus is on:

* Learning by doing
* Writing code every day
* Building mini projects instead of just watching tutorials

---

## 🧠 What I Learned

### 🔹 JavaScript Basics

* Variables (`let`, `const`)
* Data types
* Operators
* Conditionals (`if/else`)
* Functions
* Arrays
* Objects
* Loops

### 🌐 DOM Manipulation

* Selecting elements (`getElementById`, `querySelector`)
* Events (`click`, `input`)
* Changing content (`innerText`, `innerHTML`)
* Styling elements
* Creating and removing elements dynamically
